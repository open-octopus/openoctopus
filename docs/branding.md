# OpenOctopus 品牌设计

## 1. 品牌定位

| 维度 | 定义 |
|---|---|
| 产品名 | **OpenOctopus**（Open八爪鱼） |
| 品牌气质 | 聪明、多面、沉稳而有力——像章鱼一样，安静观察、多线并行、精准行动 |
| 目标受众 | 追求人生系统化治理的个人用户、一人公司创业者、知识工作者 |
| 品牌精神 | 多域掌控、持续进化、开放协作 |

## 2. 标语（Tagline）

### 2.1 主标语

- **中文**：把人生拆成多个域，让每个域都有自己的 AI 团队。
- **英文**：Organize life by areas, power each area with its own agent team.

### 2.2 短标语（社交 / Badge / 品牌露出）

| 场景 | 标语 |
|---|---|
| 技术向 | `Area-native life agent system.` |
| 行动向 | `Eight arms. Every area covered.` |
| 趣味向 | `More arms, more done.` |
| 哲学向 | `One brain. Many domains. Total control.` |
| 社区向 | `Your life, orchestrated.` |

### 2.3 口号 / 战斗口号（类似 OpenClaw 的 "EXFOLIATE!"）

**INK IT!**

> 章鱼喷墨（ink）是自我保护与快速行动的象征。
> 在 OpenOctopus 语境下，"Ink it" = 把想法落地、把计划执行、把资料沉淀。

备选：
- `REACH OUT!`（触手延伸 = 多域触达）
- `TENTACLES OUT!`（张开触手 = 全面行动）

## 3. 吉祥物

### 3.1 角色设定

| 属性 | 内容 |
|---|---|
| 名称 | **Octo**（简洁好记，呼应 Octopus） |
| 形象 | 一只拟人化的卡通章鱼 |
| 性格 | 冷静、多线程、细心、偶尔幽默，像一个靠谱的私人管家 |
| 标志动作 | 八条触手同时处理不同事务（一条拿文件、一条打电话、一条写代码……） |
| 颜色 | 深海蓝紫色系（主色），触手末端渐变为亮青色（活力点缀） |

### 3.2 性格支柱

1. **多面手** — 同时关注生活的多个领域，不遗漏任何一个
2. **沉稳可靠** — 不急不躁，持续运转，像深海中的智慧生物
3. **好奇进化** — 章鱼是无脊椎动物中最聪明的，象征持续学习与适应
4. **开放友善** — 欢迎协作，触手可以握手

## 4. 视觉体系

### 4.1 色彩方案

| 用途 | 色值 | 说明 |
|---|---|---|
| 主色 | `#1E3A5F` | 深海蓝（Deep Ocean Blue） |
| 副色 | `#6C3FA0` | 章鱼紫（Octopus Purple） |
| 强调色 | `#00D4AA` | 触手青（Tentacle Cyan） |
| 渐变 | `#6C3FA0 -> #00D4AA` | 紫到青渐变，用于触手、进度条、高亮 |
| 背景暗 | `#0D1117` | 深海夜色（Dark Mode 主背景） |
| 背景亮 | `#F6F8FA` | 浅色模式背景 |
| 文字 | `#E6EDF3` / `#1F2328` | 暗 / 亮模式文字 |

### 4.2 设计风格

- **Dark-mode 优先** — 深海主题，暗色背景为默认
- **渐变点缀** — 紫青渐变用于关键交互元素
- **圆润几何** — 章鱼的柔软身体 = 圆角、流动线条
- **矩阵布局** — Area Matrix 用网格卡片，每张卡片微微发光

### 4.3 排版

- **品牌字**：`Open` 用细体 + `Octopus` 用粗体（类似 OpenClaw 的 Open + CLAW 对比）
- **代码 / 技术**：等宽字体（JetBrains Mono / Fira Code）
- **正文**：Inter / Noto Sans SC

## 5. Logo 生图提示词

### 5.1 主 Logo（吉祥物 + 文字）

**English prompt (for Midjourney / DALL-E / Stable Diffusion):**

```
A friendly cartoon octopus mascot for a tech brand called "OpenOctopus".
The octopus is deep ocean blue-purple (#1E3A5F to #6C3FA0) with tentacle
tips that glow cyan (#00D4AA). It has large, intelligent, friendly eyes
with a calm confident smile. The octopus has 8 tentacles, each subtly
holding or interacting with different life-domain icons (a heart, a
briefcase, a coin, a book, a paw print, a dumbbell, a house, a star).
Clean vector illustration style, suitable for tech branding. Simple,
bold outlines. Dark background (#0D1117). The character should feel
smart, approachable, and capable — like a reliable life manager.
No text in image. Flat design, minimal shadows.
```

### 5.2 简版 Logo（仅吉祥物头像，用于 Avatar / Favicon）

```
A minimal cartoon octopus head icon, front-facing. Deep blue-purple
gradient body (#1E3A5F to #6C3FA0), with two large friendly intelligent
eyes and a slight smile. Two small tentacles visible at the bottom.
Glowing cyan (#00D4AA) accent on tentacle tips. Clean vector style,
flat design, no background, suitable for app icon and favicon at
small sizes (64x64, 128x128). Bold clean outlines, minimal detail.
```

### 5.3 像素风 Logo（用于 Favicon / 小尺寸场景）

```
A 16x16 pixel art octopus icon. Deep blue-purple body, cyan tentacle
tips, two white eyes with dark pupils. Minimal pixel art style, clean
and recognizable at very small sizes. Transparent background. Colors:
body #1E3A5F, accent #6C3FA0, tips #00D4AA, eyes white with #0D1117
pupils, outline #0D1117.
```

### 5.4 品牌横幅（GitHub / 社交媒体 Header）

```
A wide banner illustration for a tech project called "OpenOctopus".
Deep ocean dark background (#0D1117). A stylized octopus silhouette
in the center-left, deep blue-purple with glowing cyan tentacle tips,
reaching across the banner. Each tentacle connects to a glowing
card/node representing a life domain (work, family, finance, health,
pets, hobbies). Subtle grid pattern in the background suggesting a
matrix. Futuristic but warm. Clean tech aesthetic. No text.
Aspect ratio 3:1.
```

## 6. 生态命名体系

参考 OpenClaw 的生态命名（ClawHub、Lobster、molty.me），OpenOctopus 的生态命名建议：

| 组件 | 命名 | 来源隐喻 |
|---|---|---|
| Area 分享市场 | **AreaHub** | 域能力集散中心 |
| CLI 工具 | **tentacle** | 触手 = 命令行延伸触达 |
| Agent 网关 | **ink** | 墨汁 = 信息流通的介质 |
| 吉祥物主页 | **octo.me** | 吉祥物个人页 |
| 社区 / Discord | **The Reef**（珊瑚礁） | 章鱼栖息地 = 社区聚集地 |
| 配置 / 人格文件 | **SOUL.md** / **AREA.md** | 延续 OpenClaw 的 SOUL.md 模式 |
