## 🐙 关于 OpenOctopus

OpenOctopus 是一个 **Realm-native** 个人生活助手 Agent 系统，通过"召唤"机制将现实世界中的实体（宠物、家人、资产等）转化为有记忆、有性格的 AI Agent。

**核心概念：**
- **Realm** — 生活域（宠物、财务、健康、工作等）
- **Entity** — 域内的实体（猫咪 Luna、你的车、健身目标）
- **Summon** — 将实体召唤为有性格的 Agent
- **Living** — 激活后的智能实体，可以对话、提醒、主动建议
