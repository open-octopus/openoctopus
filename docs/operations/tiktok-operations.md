## 2. 平台特性分析

| 特性 | 说明 | 策略 |
|---|---|---|
| 算法驱动 | 不依赖粉丝量，完播率 + 互动率决定推荐 | 前 3 秒定生死，必须有强钩子 |
| 竖屏短视频 | 9:16，15s-3min 为主 | 节奏快、信息密度高 |
| 音乐/音效 | 热门音乐能蹭流量 | 配合趋势音效，但保持品牌调性 |

| 系列化 | Part 1/2/3 系列完播率高 | 所有核心概念做系列 |
| 字幕 | 85% 用户静音观看 | 每条视频必须有字幕 |

## 3. 内容策略

### 3.1 内容支柱

| 支柱 | 占比 | 说明 | 爆款潜力 |
|---|---|---|---|
| **Summon 演示** | 35% | 召唤宠物/家人/车的效果 + 多 Agent 对话 | ★★★★★ |
| **概念科普** | 25% | Realm 理念、为什么 AI 需要"域" | ★★★★ |
| **产品 Demo** | 20% | 屏幕录制 + 旁白，展示功能 | ★★★ |
| **Tech Behind** | 10% | 技术架构快讲、代码闪现 | ★★★ |
| **Meme / Skit** | 10% | 章鱼梗、AI 日常、开发者吐槽 | ★★★★★ |

### 3.2 视频格式规范

| 项目 | 规范 |
|---|---|
| 分辨率 | 1080 x 1920（9:16 竖屏） |
| 时长 | 15-60 秒（核心内容），最长 3 分钟（教程） |
| 字幕 | 必须有，白色粗体 + 黑色描边，居中偏下 |
| 品牌水印 | 右下角小 logo，低透明度 |
| 开头 | 前 3 秒必须有钩子（问题/反转/视觉冲击） |
| 结尾 | CTA：Follow for more / Link in bio / Comment below |
| 背景音乐 | 轻快电子/Lo-fi（功能演示）或热门音效（趣味向） |

## 6. 增长策略

### 6.1 冷启动

1. **Summon 宠物系列打头阵**：宠物内容在 TikTok 天然高流量
2. **蹭 AI 热点**：每次 AI 重大新闻都出评论/反应视频
3. **Duet / Stitch**：和 AI 领域创作者互动
4. **评论区营销**：在 AI/Tech 大号视频下留高质量评论
5. **跨平台导流**：YouTube Shorts 同步发布

### 6.2 持续增长

1. **系列化**：「Summon Series」「Realm of the Week」固定系列
2. **用户共创**：邀请用户拍自己的 Summon 体验
3. **挑战赛**：发起 #SummonChallenge —— 展示你想召唤什么
4. **直播**：Live coding sessions，边写代码边聊
