## 2. 账号定位

### 2.1 差异化定位

小红书上 AI 工具号很多，但绝大多数是 ChatGPT 使用技巧。OpenOctopus 的差异化：

- **不是 AI 使用教程**，而是「人生管理方法论 + AI 工具实践」
- **强调 Realm 分域思想**：每个人的人生都可以拆成多个域来治理
- **强调 Summon 召唤**：给你的宠物/家人/爱车创建 AI 分身，视觉感强

### 2.2 目标人群

| 人群 | 痛点 | 我们提供 |
|---|---|---|
| 效率工具爱好者 | 工具太多太散 | 一个系统管理所有人生域 |
| 一人公司 / 自由职业 | 需要系统化管理生活+工作 | Realm 分域 + Agent 自动化 |
| 宠物 / 家庭博主 | 宠物/家庭信息管理 | Summon 你的宠物为 AI Agent |
| 技术爱好者 | 想了解 AI Agent 前沿 | 开源、本地优先、技术深度 |

## 3. 内容策略

### 3.1 内容支柱

| 支柱 | 占比 | 说明 | 爆款潜力 |
|---|---|---|---|
| **Summon 展示** | 30% | 召唤宠物/家人/爱车为 Agent 的效果展示 | ★★★★★ |
| **Realm 方法论** | 25% | "如何用域来管理你的人生" 系列 | ★★★★ |
| **产品功能** | 20% | 功能演示、使用教程 | ★★★ |
| **AI 观点** | 15% | AI Agent 趋势、行业思考 | ★★★ |
| **开发幕后** | 10% | 开源故事、技术选型、产品思考 | ★★★ |

### 3.2 爆款内容方向

#### A. Summon 系列（最容易出圈）

```
标题：我把我家金毛「召唤」成了 AI Agent，它现在会主动提醒我打疫苗

内容：
[Momo 照片 + 界面截图]
1. 录入 Momo 的基本信息（品种、年龄、体重、喜好）
2. 点击 Summon → Momo 变成有性格的 AI Agent
3. 现在 Momo 会：
   - 提醒我该买驱虫药了
   - 高温天提醒多备水
   - 在我说"出差"时喊："谁来喂我！"
```

```
标题：给我妈创建了一个 AI 分身，从此再也不怕忘记她的喜好

内容：
1. 妈妈喜欢什么？讨厌什么？身体状况？
2. 全部录入后 Summon 为 Agent
3. 生日前它会建议："妈妈最近在学国画，送一套颜料吧"
4. 降温时提醒我："杭州今天 5°C，给妈妈打个电话"
```

#### B. Realm 方法论系列

```
标题：把人生拆成 12 个域之后，我再也不焦虑了

内容：
[八爪鱼人生治理系统信息图]
大多数人的焦虑来自——所有事情挤在一个脑袋里。

试试这个方法：
🏠 家 · 🚗 车 · 🐾 宠物 · 👨‍👩‍👦 家庭
💰 金钱 · 💼 工作 · ⚖️ 法律 · 🎯 兴趣
🏃 运动 · 🏥 健康 · 👫 朋友 · ❤️ 爱人

每个域独立管理，各有专属的知识库和 AI 助手。
这就是 Realm（域）的力量。
```

#### C. 产品功能系列

```
标题：这个开源 AI 工具，让你每个人生领域都有专属顾问团队

内容：
[功能截图/GIF]
- 法律域：有合同律师 + 劳动法顾问 + 案例分析师
- 金融域：有预算顾问 + 投资分析师 + 税务助手
- 宠物域：有健康顾问 + 你的宠物本身（召唤为 Agent）

最重要的是：全部本地运行，数据不上云。
```

### 3.3 图片/视频规范

| 类型 | 比例 | 说明 |
|---|---|---|
| 封面图 | 3:4（竖版） | 小红书首选竖版，标题大字 + 配图 |
| 多图笔记 | 3:4 | 6-9 张图，每张一个要点 |
| 视频 | 9:16（竖版） | 30-90 秒功能演示 |
| 信息图 | 3:4 | 品牌色系（深蓝紫 + 青色） |

**封面图风格：**
- 深色背景（`#0D1117`）+ 青色标题文字（`#00D4AA`）
- 或浅色背景 + 紫色标题（`#6C3FA0`）
- 必须有 Octo 吉祥物元素
- 标题字大、醒目、不超过 15 字

## 6. 增长策略

### 6.1 冷启动

1. **Summon 系列打头阵**：宠物/家人召唤为 Agent 最有传播力
2. **蹭话题**：AI 工具推荐合集、效率工具分享
3. **互动引流**：在效率工具/AI 相关笔记下留有深度的评论
4. **跨平台**：Discord 精华 → 小红书图文笔记

### 6.2 持续增长

1. **系列化**：每个系列有统一封面模板和编号
2. **合集功能**：创建"Realm 人生管理"、"Summon 召唤"、"技术幕后"合集
3. **用户共创**：邀请用户分享自己的 Realm 配置，转发到官方号
4. **节日热点**：母亲节推"给妈妈创建 AI 分身"、宠物日推 Summon 宠物
