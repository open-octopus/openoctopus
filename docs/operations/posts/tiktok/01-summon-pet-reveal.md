# TikTok #01 — Summon Pet Reveal（首发爆款）

> 时长：30-45 秒
> 风格：反转揭秘 + 产品演示
> 目标：利用宠物流量破圈，建立 Summon 概念

---

## 脚本

### 画面 + 旁白

```
[0-3s] 钩子
画面：手机屏幕上一个发光的「SUMMON」按钮
旁白："I turned my dog into an AI."
字幕：I turned my dog into an AI 🐕→🤖

[3-7s] 录入
画面：快速闪过 — 在界面上输入宠物信息（名字 Momo / 品种 / 体重 / 喜好）
旁白："First, I fed it everything about Momo."
字幕：Name: Momo | Breed: Golden Retriever | Weight: 38kg

[7-10s] 召唤
画面：手指点击「SUMMON」按钮 → 屏幕闪光 + 粒子特效
音效：魔法音效 / 能量充能声
旁白："Then I hit Summon."
字幕：✨ SUMMON! ✨

[10-18s] Momo 活了
画面：聊天界面出现，Momo Agent 的头像亮起
旁白："Now Momo is a living AI agent. It remembers everything. It has personality. And it talks back."
字幕：Momo is alive. Memory ✓ Personality ✓ Initiative ✓

[18-28s] 对话演示
画面：展示几条对话（打字效果逐条出现）
  用户："I'm traveling next week."
  Momo："WHO WILL FEED ME FOR 5 DAYS?! 😤"
  Momo："Last time you left, the neighbor forgot my snacks!"
  用户："Mom said she'll come."
  Momo："...fine. But tell her I want the chicken treats, not the fish ones."
旁白：跟读对话内容
音效：每条消息出现时有气泡音效

[28-35s] 跨域协作
画面：多个 Agent 同时弹出回复
  👩 Mom: "I'll take care of Momo."
  🚗 Car: "Charge to full?"
  💰 Budget: "Trip estimate: $800"
旁白："And it's not just Momo. Every realm of your life responds."
字幕：Cross-realm intelligence 🐙

[35-42s] 收尾 CTA
画面：OpenOctopus logo + GitHub 地址
旁白："OpenOctopus. Open source. Your data, your tentacles."
字幕：Open source 🐙 Link in bio
音效：结尾标志音
```

### 配乐建议

- 主体部分：轻快电子节拍（如 CapCut 自带的 "Tech Vibes" 类）
- 召唤时刻：低音 drop + 魔法音效
- 对话部分：降低音乐音量，突出消息音效

## 视频制作提示词

### 方案 A：AI 生成视频片段（Runway / Pika / Sora）

#### 召唤特效片段（7-10s 使用）

```
A smartphone screen showing a glowing cyan button labeled "SUMMON" on a dark interface. A finger taps the button. The screen erupts with cyan (#00D4AA) and purple (#6C3FA0) particle effects, energy waves rippling outward from the button. The particles swirl and coalesce into the silhouette of a golden retriever dog. Dark background. Cinematic, magical transformation effect. Vertical video 9:16 ratio. 3 seconds duration.
```

#### Momo Agent 激活片段（10-12s 使用）

```
A dark-themed chat interface on a phone screen. An avatar circle lights up with a golden glow — inside is a golden retriever dog photo. Below the avatar, the name "Momo" appears with typing indicators (three bouncing dots). Then a chat bubble slides in from the left with text. The interface has cyan (#00D4AA) accent colors on a dark (#0D1117) background. Vertical 9:16 ratio. 3 seconds.
```

#### 多域响应片段（28-35s 使用）

```
A phone screen showing a dark chat interface. Multiple chat bubbles appear rapidly from different agents — a dog icon, a woman icon, a car icon, a piggy bank icon — each sliding in with a subtle bounce animation. Each bubble has a different colored accent (cyan, pink, blue, green). The bubbles stack vertically creating a lively multi-agent conversation. Vertical 9:16 ratio. 4 seconds.
```

### 方案 B：屏幕录制 + 后期特效

如果产品已有原型/Demo，直接屏幕录制 + 用 CapCut 后期加特效：

1. **录制工具**：ScreenStudio（Mac）或 OBS
2. **录制内容**：实际操作流程（录入 → Summon → 对话）
3. **后期特效（CapCut）**：
   - 召唤时刻加「光效」转场
   - 每条消息出现加「弹入」动画
   - 背景加轻微粒子飘浮
   - 全程自动字幕 + 手动校准
4. **配音**：自己录制或用 ElevenLabs 生成英文旁白

### 方案 C：混合（推荐）

- 实拍：人 + 狗的真实镜头作为开头钩子
- 屏录：产品操作部分
- AI 生成：召唤特效、概念可视化片段
- CapCut 合成 + 字幕 + 配乐
