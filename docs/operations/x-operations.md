## 4. 互动规范

### 4.1 回复策略

| 场景 | 回复风格 |
|---|---|
| 用户提问 | 简洁直接回答 + 链接到文档 |
| 用户 showcase | 🐙 + 赞美 + "Share in The Reef!" |
| Bug 反馈 | 感谢 + 引导到 GitHub Issue |
| 竞品比较 | 不贬低他人，聚焦自身差异化（Realm + Summon） |
| 负面评价 | 理性回应，承认不足，展示改进计划 |

### 4.2 语调

- 直接、技术自信、偶尔幽默
- 不用 "revolutionary" "game-changing" 等营销词
- 可以用的表达：`Your data, your tentacles.` / `SUMMON!` / `Eight arms. Every realm covered.`
- 中英文混发（跟随受众）

## 5. 增长策略

### 5.1 冷启动（0 → 1K followers）

1. **#BuildInPublic**：每天分享开发进展，截图 + 思考
2. **回复 AI/Agent 领域大 V 的推文**：有深度的技术评论
3. **Thread 长文**：写 3-5 条高质量 Thread（如 "Why your AI assistant needs Realms"）
4. **GitHub 联动**：每个 Release 在 X 同步发布
5. **Discord → X**：社区精华内容搬运到 X

### 5.2 稳步增长（1K → 10K）

1. **定期 Demo 视频**：30-60 秒功能演示 GIF/视频
2. **用户故事**：邀请早期用户分享使用体验
3. **技术 Thread**：架构、设计决策、踩坑分享
4. **跨平台引流**：Product Hunt launch 时 X 配合推广
5. **活动**：Summon of the Month 评选结果在 X 公布
